## PRODUCT API

### INSET PRODUCTS DATA
### POST /api/products
#### Payload
```JSON
{
    "name": "My Product from Postman",
    "category": "Tool",
    "price": 0,
    "tags": "test1"
}
```
------------

### UPDATE PRODUCTS DATA BY ID
### PUT /api/products/:id
#### Payload
```JSON
{
    "name": "My Product from Postman",
    "category": "Tool",
    "price": 0,
    "tags": "test1"
}
```
------------
